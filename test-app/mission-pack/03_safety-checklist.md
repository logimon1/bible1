# 야간 숲 안전 점검표

## 시작 30분 전

- [ ] 일몰 후에도 보이는 이동로를 확보했다.
- [ ] 돌, 뿌리, 계단, 비탈을 표시했다.
- [ ] 스테이션마다 LED 랜턴이 있다.
- [ ] 실제 불, 폭죽, 연막, 날카로운 소품을 사용하지 않는다.
- [ ] 화장실과 집결 장소 방향 표지가 있다.
- [ ] 비상 연락망과 보호자 연락 방법을 진행자들이 알고 있다.
- [ ] 구급가방과 생수 위치를 모두 알고 있다.
- [ ] 기상 악화 시 실내 대체 장소를 정했다.

## 참가자 입장 전

- [ ] 팀별 인원과 짝을 확인했다.
- [ ] 중단 신호 `빛`을 모두가 이해했다.
- [ ] 뛰지 않기, 밀지 않기, 눈 가리지 않기를 안내했다.
- [ ] 불안하거나 어지러운 사람은 즉시 빠질 수 있다고 안내했다.
- [ ] 휴대폰 손전등을 비상용으로 켜는 방법을 안내했다.

## 진행 중

- [ ] 스테이션 사이 이동 때 선두와 후미가 모두 보인다.
- [ ] 팀이 표시된 안전선 안에 있다.
- [ ] 진행자는 참가자를 만지거나 뒤쫓지 않는다.
- [ ] 미션이 길어지면 중단하고 보상 처리한다.
- [ ] 누가 `빛`이라고 말하면 조명을 켜고 전원 정지한다.
- [ ] 이동 때마다 팀별 인원 수를 센다.

## 즉시 중단 기준

다음 중 하나라도 해당하면 미션을 중단하고 밝은 집결 장소로 이동합니다.

- 천둥, 강한 비, 낙뢰 예보
- 길이 미끄러워진 경우
- 참가자가 길을 잃었거나 짝과 떨어진 경우
- 공황, 호흡 곤란, 구토, 심한 어지럼
- 조명 또는 통신 장비 고장

## 종료 후

- [ ] 모든 팀이 집결 장소에 도착했다.
- [ ] 입장 인원과 퇴장 인원이 같다.
- [ ] QR 보상 처리가 끝났다.
- [ ] 소품, 랜턴, 카드, 테이프를 회수했다.
- [ ] 불편하거나 무서웠던 참가자의 의견을 짧게 들었다.
