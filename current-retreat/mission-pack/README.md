# THE WAR 밤 숲 미션 팩

이번 수련회 현장에서 사용하는 야간 숲 미션 자료입니다. 기존 앱의 6개 미션 QR 경로와 바로 연결되도록 구성했습니다.

## 파일 구성

- `01_facilitator-guide.md`: 진행자용 전체 흐름, 멘트, 역할, 채점
- `02_mission-cards.md`: 참가자에게 읽어 주거나 현장 카드로 인쇄할 짧은 원고
- `03_safety-checklist.md`: 입장 전, 진행 중, 종료 후 안전 점검표
- `04_materials-list.md`: 스테이션별 준비물과 대체 준비물
- `data/night-forest-missions.json`: 앱/QR/인쇄물 연동용 원본 데이터
- `scripts/build_mission_pack.py`: 진행자 가이드와 미션 카드 PDF 생성기
- `output/pdf/night-forest-mission-pack.pdf`: 인쇄용 통합 PDF

## 운영 콘셉트

공포 수위는 5점 만점에 2점입니다. 숲의 어둠, 낮은 조명, 속삭이는 내레이션으로 긴장감을 만들되 참가자를 갑자기 놀라게 하거나 혼자 남겨 두지 않습니다.

- 모든 이동은 표시된 안전선 안에서만 합니다.
- 어두운 구간에서도 뛰지 않습니다. 달리기 미션도 걷기 또는 빠른 걸음으로 대체합니다.
- 눈을 가리거나, 사람을 밀거나, 높은 곳에 오르거나, 불을 사용하지 않습니다.
- 중단 신호는 `빛`입니다. 누구든 말하면 즉시 조명을 켜고 진행을 멈춥니다.
- 각 스테이션은 최소 성인 1명, 전체 숲에는 총괄 1명을 둡니다.
- 참가자는 2명씩 짝을 유지하고, 스테이션 사이 이동 때 인원수를 확인합니다.

## 앱 연결

핵심 미션 QR은 아래 경로를 그대로 사용합니다.

| 파트 | QR 경로 | 미션 카드 |
| --- | --- | --- |
| 진리의 허리띠 | `/mission/judgment` | 속삭이는 이정표 |
| 의의 호심경 | `/mission/endurance` | 흔들리는 늪 |
| 복음의 신발 | `/mission/speed` | 발자국의 전령 |
| 믿음의 방패 | `/mission/teamwork` | 소리 없는 방패 |
| 구원의 투구 | `/mission/intellect` | 투구 속 암호 |
| 말씀의 검 | `/mission/power` | 검을 세우는 결단 |

성공 확인은 앱의 `미션 완료 · 보상 받기` 버튼으로 처리합니다. 진행자가 성공을 확인하기 전에는 참가자가 보상 버튼을 누르지 않도록 안내합니다.

## 권장 진행 시간

- 입장과 안전 브리핑: 8분
- 핵심 미션 6개: 스테이션당 6-8분
- 히든 미션: 5분
- 최종 보스: 8분
- 퇴장과 인원 확인: 5분

실제 야외 동선과 날씨에 따라 스테이션 수를 줄여도 됩니다. 밤에 처음 운영하는 경우에는 핵심 미션 4개만 선택하고, 나머지는 낮 리허설에서 테스트하는 것이 안전합니다.
