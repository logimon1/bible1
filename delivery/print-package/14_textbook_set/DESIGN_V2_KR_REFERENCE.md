## V2에 적용할 공통 원칙

- 대상은 중1~고3으로 명확히 고정한다. 초등 고학년판은 이후 별도 판본으로 설계한다.
- B5 176×250mm는 납품용 1과 워크북의 활동성과 기록 공간을 위한 프로젝트 사양으로 유지한다. 다만 국내 청소년 학생용의 단일 표준 판형이라고 표현하지 않는다. 2026 GPL-S 학생용 128×187mm, 하나 바이블 학생용 148×210mm, 드림웨이브 청소년용 140×196mm처럼 학생책은 A5 안팎의 소형 사례가 많고, 교사용은 GPL-S 187×256mm·하나 바이블 182×257mm처럼 더 크게 설계되는 경향이 확인된다.
- 첫 장은 제작 라벨이 아니라 실제 표지여야 한다.
- 매 펼침은 하나의 학습 행동에 집중한다: 장면 보기 → 본문 관찰 → 핵심 이해 → 분류 활동 → 개인 적용 → 주간 실천.
- 본문 설명은 2단 이하, 한 줄 길이는 짧게 유지하고 학생 본문은 11pt 이상을 기본으로 한다.
- 색은 장식이 아니라 기능으로 사용한다.
  - cobalt: 말씀·기준
  - yellow: 관찰·질문
  - coral: 멈춤·주의
  - teal: 선택·실천
- 1과의 시각 모티프는 `엉킨 생각을 곧은 진리선으로 정리하기`이다. 선, 카드, 단계 번호를 학생책·교사용·슬라이드·활동물에 공통 적용한다.
- 학생용과 교사용은 같은 세계관을 공유하되, 교사용은 시간·멘트·예상 반응·이어갈 질문·주의 신호가 빠르게 스캔되도록 2단 정보 구조를 사용한다.
- 활동카드는 저잉크 사무용 인쇄를 기준으로 하며, 카드 크기와 분류판의 실제 물리적 호환을 검증한다.
- QR은 실제 납품 URL이 확정되기 전에는 가짜 QR을 넣지 않는다. 학생 화면에 내부 코드명도 노출하지 않는다.
- 슬라이드는 제목 36~44pt, 본문 24pt 이상을 원칙으로 하고 한 화면에 한 메시지만 둔다.

## 폐기하는 V1 패턴

- 모든 페이지에 같은 제목줄과 같은 메모 상자를 반복하는 방식
- 페이지 제목에 `5쪽 - 도입`, `앞면 -` 같은 제작 지시를 노출하는 방식
- 교사용 하단 절반을 무조건 메모칸으로 비우는 방식
- 진한 녹색·금색의 성인용 카드게임 인상
- 슬라이드 60% 이상을 목적 없이 비우고 작은 글을 좌상단에 모으는 방식
- 실제 카드보다 작은 분류판
- 확정되지 않은 URL의 QR 또는 내부 미션 ID 노출
