# 중·고등부 4주 전신갑주 공과 세트

이 폴더는 교회 납품용 공과책 마스터 영역입니다. 1과 `진리로 중심을 잡아라`는 국내 청소년 공과 교재 레퍼런스를 기준으로 전면 재설계한 V2.0 확정본이며, 2~4과는 이후 개발 대상입니다.

## 상품 정의

- 대상: 중학교 1학년~고등학교 3학년 소그룹
- 기간: 주 1회, 4주
- 차시: 45~60분, 기본안 55분
- 핵심 본문: 에베소서 6:10~18
- 상품 방식: 교회별 전용 앱 + 교회별 전용 DB + 인쇄 세트
- 콘텐츠 원본: 월간 4주판
- 파생 상품: 1~3일 수련회 압축판

초등부는 이 청소년용 템플릿을 재사용하지 않고 판형·색채·활동 밀도를 별도로 설계합니다. 사용할 성경 번역본과 교단별 표현은 실제 납품 교회와 확정하며, 학생책은 성경 본문 전문을 싣지 않고 교회에서 사용하는 성경을 직접 읽도록 구성합니다.

## 주요 파일

- `DESIGN_V2_KR_REFERENCE.md`: 2025~2026 국내 제품 조사와 V2 편집 기준
- `product_spec.md`: 상품 구성·판형·출시 기준
- `curriculum.md`, `curriculum.json`: 4주 커리큘럼과 구조화 데이터
- `student/lesson01_truth.md`: 학생책 1과 8쪽 V2 원고
- `teacher/lesson01_truth_teacher.md`: 교사용 1과 12쪽 V2 원고
- `activities/lesson01_truth_cards.md`: 활동카드 12장 원고
- `activities/lesson01_home_connection.md`: A6 가정연계 원고
- `teacher/lesson01_slides.md`: 16:9 슬라이드 10장 원고
- `art/illustration_briefs.md`: 일러스트 프롬프트·제작 기록
- `art/generated/lesson01_truth_editorial_v2.png`: 대표 장면
- `art/generated/lesson01_truth_listening_spot_v2.png`: 듣기 보조 장면
- `scripts/build_lesson01_v2.py`: V2 PDF 생성기
- `scripts/verify_lesson01.py`: 페이지·문구·도련·PPI·앱 정합성 검증기
- `output/pdf/`: V2 PDF 5종과 manifest

## 1과 재생성·검증

```bash
npm run build:lesson01
npm run verify:lesson01
```

생성물은 다음과 같습니다.

- 학생책 B5 인쇄용 8쪽
- 교사용 지도서 A4 12쪽
- 활동카드·분류판·정답 A4 8쪽
- 가정연계 카드 A6 양면 2쪽
- 교사용 슬라이드 16:9 10쪽

같은 파일과 manifest가 `11_print_ready/`에도 납품용으로 복사됩니다.

## 1과 V2 상태

- 원고·편집·일러스트·PDF 재생성 구조 확정
- 총 40쪽 전 페이지 렌더 검수 완료
- 카드 완성 크기 63×88mm, A4 2×3 양면 배치와 A4 2×2 분류판 확정
- 실제 URL 미확정 상태의 가짜 QR과 내부 미션 ID 미노출
- 교회명, 번역본, 실제 납품 URL QR, 보호 담당자는 납품 시 개인화

상세 확정 기록은 `LESSON01_FINAL.md`를 확인합니다.

## 다음 제작 순서

1. 1과를 실제 중·고등부 6~12명 소그룹에서 파일럿합니다.
2. 납품 교회의 신학·교육·보호 기준 피드백을 기록합니다.
3. V2 디자인 시스템으로 2~4과 원고와 시각 자산을 제작합니다.
4. 과별 대표 일러스트와 반복 등장인물 시트를 확정합니다.
5. 전체 4과의 학생책·교사용·활동물·가정연계·슬라이드를 합본 교정합니다.
6. 실제 교회 URL로 QR을 생성해 휴대폰·인쇄물 리허설을 진행합니다.

## 전체 세트 완료 정의

- 신학·교육·맞춤법·인쇄 교정
- 실제 학생 파일럿과 실제 휴대폰 QR 리허설
