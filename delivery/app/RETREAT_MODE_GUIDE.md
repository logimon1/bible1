# RETREAT MODE GUIDE

## 목적

1~3일 수련회에서 QR 미션, 장비 수집, 교환소, 히든 QR, 보스전을 빠르게 운영하는 모드입니다.

## 설정

```json
{
  "programMode": "retreat"
}
```

또는 Vercel 환경변수:

```text
PROGRAM_MODE=retreat
```

## 추천 운영 순서

1. 입장 QR 또는 기본 URL로 학생 생성
2. 팀별 QR 미션 진행
3. 장비뽑기와 장비 수집
4. 히든 QR 공개
5. 교환소 1/2 운영
6. 갑주 도전 또는 보스전
7. 팀 랭킹/개인 랭킹 확인

## 교환소

- `/exchange/1`, `/exchange/2` QR을 출력합니다.
- 한 교환소에는 동시에 2명만 들어갑니다.
- 각자 최대 2개 장비를 제시합니다.
- 양쪽이 확인하면 서버가 동시에 교환 처리합니다.
- 문제가 생기면 관리자가 교환소를 초기화합니다.

## 보스전

`/boss`는 수련회 마지막 단계 QR로 사용합니다. 실제 오픈 조건은 운영자가 설명하고, 앱에서는 보스전 안내와 장비 상태를 보여주는 용도로 사용합니다.

## 현장 체크

운영 전 반드시 확인:

- 학생 2명으로 QR 완료 테스트
- 같은 QR 중복 접속 테스트
- 교환소 완료 테스트
- 관리자 수동 지급 테스트
- 랭킹 반영 확인
- QR 출력물 카메라 인식 확인
